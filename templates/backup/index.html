Safeguard your LNbits data with automatic scheduled database backups. Schedule backups hourly, daily, weekly, or monthly with retention policies to automatically manage storage. Features include SQLite and PostgreSQL support, optional gzip compression, error tracking with detailed status monitoring, and manual backup triggers for on-demand protection.

How to use Database Backup:

  1. Create a new backup schedule by clicking "New Backup Schedule"
  2. Enter a descriptive name to identify the schedule
  3. Specify the backup directory path (each schedule gets its own subdirectory)
  4. Choose the frequency (hourly, daily, weekly, or monthly)
  5. Set the start date and time (this determines when backups run, e.g., daily at 2:00 AM)
  6. Configure retention count (number of backups to keep)
  7. Enable compression to save disk space
  8. Save and activate the schedule

Backups will automatically run according to your schedule. You can manually trigger a backup anytime, pause, edit, or delete schedules as needed.
